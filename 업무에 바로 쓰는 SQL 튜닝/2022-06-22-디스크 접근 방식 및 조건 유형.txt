## 디스크 접근 방식

### 시퀀셜 엑세스
- 인접한 페이지를 차례대로 읽는 디스크 접근 방식이며, 테이블 풀 스캔시 활용
- 디스크 헤더의 움직임을 최소화하여 리소스 절약
- 테이블 풀 스캔이기에 인접한 페이지를 여러개 읽는 "다중 페이지 읽기"방식으로 수행

### 랜덤 액세스
- 페이지의 물리적 위치를 고려하지 않고, 임의대로 페이지에 접근하는 디크스 접근 방식.
- 다중 페이지 불가능, 디스크 헤더의 불규칙적인 움직임으로 인한 데이터 접근 시간이 오래 걸림
- 최소한의 페이지에 접근할 때 사용

## 조건 유형

### 엑세스 조건
- 디스크에서 데이터를 검색하는 조건

### 필터 조건
- 디스크에서 꺼내온 데이터를 가공하는데 쓰이는 조건
- 액세스 조건에 따라 디스크에서 가져온 데이터를 필터링 하는 기준
- 필터링할 데이터가 없을수록 좋은 SQL문이고 반대의 경우 튜닝이 필요하다
- 필터 조건으로 제거될 데이터라면 애초에 스토리지 엔진에서 제외되는 편이 성능적으로 좋다.
